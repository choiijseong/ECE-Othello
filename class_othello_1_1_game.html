Detailed Description

Classe de Jeu.

Classe initiatrice de jeu. Elle gère directement tous ses aspects, les appels à l'interface, le passage des joueurs, etc... Il suffit d'appeler le constructeur en fournissant les paramètres nécessaires. Le constructeur quitte dès que la partie est terminée.
